Featured Project

Retro Secrets Online Jewellery Shop

A responsive front-end web app of an online jewellery shop made with React, styled with Sass, and deployed on Heroku.

  • Responsive

    This web app is responsive across devices of all sizes and supports major browsers including Google Chrome, Firefox, Safari 13+, Opera, and Edge.

  • Scalable

    This project is coded in small and re-usable React function components and styled with Sass, making it easy to scale.

  • Fast loading

    As a single-page React app that pulls data from a mock API service, this web app is fast to load.

  • With UX in mind

    This web app makes it easy for users to view products, adjust and delete items in the cart, and change shipping options no matter what device they are using.

Weather app

Get current, hourly, and daily weather forecasts of your current location and any place you search for.

React weather app

A live responsive web app made with React function components and hooks, styled with Sass, and deployed on Heroku.

Features

  • Automatically loads weather forecasts of the user's current location
  • Automatically completes the user's location search
  • Supports city name, address, or postcode as search terms
  • Displays current weather, 24-hour hourly forecasts, and 8-day daily forecasts
  • Customised background images that indicate different weather conditions and time of the day

APIs

  • Google Geocoding API
  • Google Places API
  • Open Weather App API
  • Geolocation API

Libraries

  • React
  • React Google Places Autocomplete
